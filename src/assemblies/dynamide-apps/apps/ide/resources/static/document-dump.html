document-dump test

This page is a test of using document.documentElement, which returns the HTML element. This page has some representative elements that cover what is allowed in HTML. Note that the includes in the head point at the /dynamide/ide project, which requires that the IDE be run at least once, or the scripts won't be in cache.

On document onload we then call our dynamide function domToString(window.document) which wraps the HTML tags for us. domToString() calls documentInnerHTML(window.document) to spit out the entire DOM, minus the HTML tag itself, as a string.
