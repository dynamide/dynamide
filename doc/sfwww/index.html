:: Source Files
The project is in active development. Please check out the files from sourceforge.net using the development page, or sourceforge's git browser or

or directly with svn access: or directly with git access: 

 git clone git://git.code.sf.net/p/dynamide/dynamide dynamide
or from github. (We consider github to be "origin", although we push to both sourceforge and github in sync.) 

  git clone  https://github.com/dynamide/dynamide.git
Note that you can still get to our svn repository, but it is for version history for build-92 and earlier:
       svn co https://dynamide.svn.sourceforge.net/svnroot/dynamide dynamide
And you can use sourceforge's svn browser
